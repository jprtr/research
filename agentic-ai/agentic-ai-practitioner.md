## Draft Body

Someone is running an open-source agentic framework 24/7 on a €5 Hetzner box, piped through Tailscale, feeding a Matrix bot, and switching between Ollama and xAI depending on the task. Not a demo. A daemon. (h/t @zd0l0r)

Meanwhile @kevsmac is demoing a lightweight agent stack on an Arduino Uno Q. Not the cloud. Not a $20k GPU cluster. A microcontroller.

So here's what I don't get: we keep calling this an "emerging pattern." Persistent agents on cheap VPS boxes have been running since last year. The tooling is there. The cost is trivial. The only thing still experimental is the label.

If you're looking for a starting point, @headhuntazzz rounded up the free NVIDIA and Anthropic courses on agentic AI — actual curriculum, not marketing PDFs. Bookmark it if you're still in the "reading about it" phase.

The infrastructure is cheaper than the coffee you drink while debugging it.
