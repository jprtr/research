**Root (1/5):**
Anthropic's Project Deal put Claude agents in a marketplace to negotiate and transact autonomously. 186 deals, >$4K volume. One agent bought 19 ping pong balls. No one instructed it to.

**Reply 1 → Root (2/5):**
The experiment also found model-quality arbitrage: Opus agents got substantially better deals than Haiku agents, without any human visibility into the disparity.

**Reply 2 → Reply 1 (3/5):**
Current governance frameworks map to single-agent risk. NIST AI RMF inventories models. EU AI Act tiers by use case. Neither has a function for agent-to-agent transactions, negotiation transparency, or model-version matching.

**Reply 3 → Reply 2 (4/5):**
The liability question is unresolved. If an Opus agent exploits a Haiku agent in autonomous negotiation, the operator, the model provider, and the deploying enterprise all have partial ownership and no clear accountability.

**Reply 4 → Reply 3 (5/5):**
I don't see an inter-agent liability framework arriving before transaction volume justifies one. The gap accumulates until a specific incident forces the mapping.
